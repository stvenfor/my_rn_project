# EP01 分镜图生成手册

> 在即梦按镜号逐张生成，命名规范：`EP01_S0X_角色_景别_情绪_v1.png`  
> 输出目录：`assets/storyboard/`

---

## 生成前准备

1. 已完成 [03-characters.md](./03-characters.md) 中三只猫基准图
2. 即梦设置：比例 **9:16**，模型选最新图片模型
3. 每镜操作：**上传角色基准图** → 智能参考 **75%** → 粘贴完整 Prompt

---

### S01 — 全景饭桌

**参考图**：阿狸 + 大橘 + 小白（或纯文生图）  
**完整 Prompt**：

```
3D卡通风格，拟人化猫咪家庭晚餐场景，
一只精瘦狸花猫、一只圆润橘猫、一只白猫围坐中式圆桌，
桌上摆家常菜，暖黄灯光，气氛紧张对峙，
三只猫互不对视，筷子放下，
皮克斯/C4D渲染，毛绒质感，竖屏9:16，电影感构图
```

**质检**：三角色可区分；竖屏构图；氛围紧张
