# EP01 剪映剪辑与配音时间线

> 工具：剪映 PC / App | 画幅 9:16 | 总时长目标 50–60s

---

## 一、音轨结构

```
轨道1: 视频/图片（S01-S08）
轨道2: 配音（多角色 TTS 或真人）
轨道3: BGM（30-40% 音量）
轨道4: 音效（拍桌、心跳、咚、乌鸦）
轨道5: 字幕
```

---

## 二、角色声线配置

| 角色 | 剪映音色建议 | 语速 | 音调 |
|------|-------------|------|------|
| 阿狸 | 「成熟男声」/「知性女声」偏冷 | 1.1x | 偏低 |
| 大橘 | 「温暖男声」/「少年音」 | 1.0x | 偏高，句尾颤 |
| 小白 | 「活泼女声」/「调侃男声」 | 1.05x | 中等 |

**操作**：文本 → 新建文本 → 粘贴台词 → 文本朗读 → 选择音色 → 生成

## 四、音量混音

| 轨道 | 音量 | 备注 |
|------|------|------|
| 配音 | 75–80% | 主音轨 |
| BGM | 30–40% | 不盖台词 |
| 音效 | 60–80% | 咚、心跳要突出 |

**操作**：选中音频 → 音量 → 调节；或使用「音频分离」后单独调节

---

## 五、字幕规范

- 字体：思源黑体 / 站酷快乐体（可爱风）
- 毒句：`倒插门` `赘婿` `工资卡` `37度` `心肌梗塞` → 加大 1.2x 或变色（黄/红）
- 位置：底部安全区，避开抖音 UI
- 每句不超过 15 字，长句拆分

---

## 六、包装清单

| 元素 | 建议 |
|------|------|
| BGM | 抖音搜「家庭伦理」「悬疑短剧」纯音乐；或从同类爆款分离音频 |
| 滤镜 | 统一「暖食」或「质感」滤镜，全片一致 |
| 转场 | 硬切为主，S05 句间可用闪白 |
| 导出 | 1080×1920，30fps，H.264 |

---

## 七、导出前检查

- [ ] 总时长 50–60s
- [ ] 每 2–3s 有信息增量
- [ ] 配音清晰，BGM 不盖台词
- [ ] S07 咚 + 心肌梗塞字幕到位
- [ ] 字幕无错别字
- [ ] 已添加 AIGC 标识（若平台要求）
