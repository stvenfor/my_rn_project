# EP01 动态化指南

> **推荐方案：A（静态图 + 轻动效）** — 适合首次出片，台词驱动，1 小时内可完成。  
> 方案 B 为进阶升级，出片后可逐镜替换。

---

## 方案 A：静态图 + 轻动效（MVP 推荐）

### 原理

对话型短剧的核心是**台词与切镜节奏**，评论讨论「说了什么」多于画面动态。静态分镜 + 剪映动效即可达到赛道平均水平。

### 素材导入

1. 将 `assets/storyboard/` 下 8 张分镜图按 S01–S08 顺序导入剪映
2. 画幅：9:16，分辨率 1080×1920
3. 每镜默认时长见下表

| 镜号 | 默认时长 | 动效 | 参数建议 |
|------|---------|------|---------|
| S01 | 3s | 缓慢缩放 | 缩放 100%→105%，缓入 |
| S02 | 5s | 固定 | 无 |
| S03 | 5s | 推近 | 缩放 100%→110% |
| S04 | 8s | 固定 | 无 |
| S05 | 10s | 轻微推近 + 闪白（句间） | 每句毒舌加 0.1s 闪白 |
| S06 | 5s | 震动 | 幅度 3–5，持续 0.5s |
| S07 | 4s | 关键帧倒放 / 缩放 | 100%→120% 模拟倒倒 |
| S08 | 3s | 定格 | 最后一帧冻结 |

### 剪映操作步骤

1. **导入图片**：媒体 → 导入 8 张分镜 → 拖入时间线
2. **设置时长**：选中片段 → 拖动边缘至目标秒数
3. **添加动效**：选中片段 → 动画 → 入场/组合 → 「渐显」「缩放」
4. **S07 躺倒**：若静态图不够夸张，复制 S06 崩溃图，加「旋转」关键帧模拟后倒，或叠加「倒放」+ 音效
5. **S05 闪白**：在句间插入 0.1s 白场（调节 → 亮度拉满）或「闪白」特效

### 方案 A 产出检查

- [ ] 每 2–3 秒有信息增量（新台词/新表情）
- [ ] S07 有「咚」倒地音效
- [ ] 全片滤镜统一（暖色/略降饱和）

## 方案 B：图生视频（进阶）

### 适用场景

- 需要更强「短剧感」
- 计划系列化 IP，提升观感
- 已熟悉即梦/可灵操作

### 工具

- 即梦 Seedance 2.0 / 视频 3.0 Pro
- 可灵 Kling（复杂动作备选）

### 逐镜图生视频 Prompt

| 镜号 | 图生视频 Prompt | 时长 |
|------|----------------|------|
| S01 | 镜头缓慢横移扫过三只猫，气氛凝固，轻微推近，动作幅度小 | 3s |
| S02 | 橘猫微微前倾说话，表情委屈，镜头固定中景 | 5s |
| S03 | 狸花猫嘴角缓慢上扬，眼神斜视，镜头缓慢推近 | 5s |
| S04 | 白猫嗑瓜子，左右看看，镜头固定 | 5s |
| S05 | 狸花猫连续说话，头部微动，镜头缓慢推近，动作不超过2个 | 8s |
| S06 | 橘猫捂胸向后退半步，表情崩溃，镜头固定 | 5s |
| S07 | 橘猫向后倒躺倒，动作夸张，镜头轻微震动 | 4s |
| S08 | 狸花猫面无表情，轻微眨眼，镜头定格 | 3s |

### 图生视频规范

- 每段 **5–10 秒**，动作指令不超过 2 个
- 上传该镜首帧图 → 选择「图生视频」
- 失败只重生成单镜，不整片重来
- 生成后导出，替换时间线中对应静态图

### S07 躺倒进阶

评论「躺地上后再来个尿失禁就完美了」→ 可选夸张化：

```
橘猫向后倒躺倒，夸张喜剧，可选手部捂腹，镜头轻微震动
```

或剪映：S06 崩溃图 + 关键帧 Y 轴位移模拟倒倒 + 咚音效（方案 A 已足够）
